### Development

1.  **Clone the repository.**

    ```
    $ git clone https://github.com/AmberThrall/portfolio
    ```

1.  **Install the dependencies**

    ```
    $ npm install
    ```

1.  **Start developing**

    ```
    $ cd portfolio && gatsby develop
    ```
    The site is now running at http://localhost:8000.

1.  **Publish**

    Commit and push the changes to GitHub. Netlify will see the update and build the latest site.
